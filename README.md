
# Gifted Bulk STK Push

1. Copy .env.example to .env
2. Add your API key.
3. npm install
4. npm start

## GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin YOUR_REPO_URL
git push -u origin main

## Render
Create Web Service from GitHub repository.
Build: npm install
Start: npm start
Set GIFTED_API_KEY environment variable.
